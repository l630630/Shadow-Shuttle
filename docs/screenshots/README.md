## 📝 截图规范

请参考 [SCREENSHOTS_GUIDE.md](../SCREENSHOTS_GUIDE.md) 了解详细的截图规范和拍摄指南。

### 基本要求

- **格式**: PNG
- **桌面截图**: 1920x1080
- **移动端截图**: 1170x2532 (iPhone 14 Pro)
- **架构图**: 2000x1500 或更高
- **命名**: 小写字母，连字符分隔

## 🎯 如何添加截图

1. 按照 [SCREENSHOTS_GUIDE.md](../SCREENSHOTS_GUIDE.md) 拍摄截图
2. 将截图保存到对应的子目录
3. 更新本文件的状态列表
4. 提交到 Git 仓库
