# 影梭 (Shadow Shuttle) - AI 驱动的远程设备管理平台

**项目周期：** 2024年 10月 - 2024年 12月

## 项目简介

基于 React Native 开发的跨平台移动应用，结合 WireGuard VPN 和 AI 自然语言处理技术，实现通过手机对远程设备的智能化管理。用户可使用自然语言指令远程操作电脑，无需记忆复杂命令行语法。

## 技术栈

React Native、TypeScript、Zustand、Go、gRPC、WireGuard、SSH、OpenAI/Gemini/Claude API、ESLint + Prettier、Git

## 核心职责

### 🎨 架构设计
- 负责移动端整体架构设计，规划 UI 组件拆分、设计业务组件、组织复用逻辑（如 Composition API）
- 负责代码框架设计，规划 UI 组件拆分、设计业务组件、组织复用逻辑（如自定义 Hooks）
- 统一 API 的设计，封装 AI Service API、使用 Zustand 实现全局数据状态管理，确保数据高效流转

### 🤖 AI 自然语言系统
- 独立承担 AI 自然语言处理模块开发，集成多个 AI 提供商（OpenAI、Gemini、Claude、硅基流动）
- 实现智能命令解析，将自然语言转换为可执行的 Shell 命令
- 设计安全检查机制，识别危险命令并要求确认，实现隐私过滤保护敏感信息

### 🔐 安全与体验
- 实现生物识别认证、SSH 密钥管理、设备 QR 码配对等安全机制
- 设计聊天式交互界面，实现命令历史/收藏、终端输出优化、设备状态监控
- 集成 WireGuard VPN、实现 SSH 连接管理、设计设备发现服务

### 📊 性能优化
- 使用 Zustand 实现全局状态管理，数据持久化，优化应用启动速度
- 解决设备同步问题，优化 AI 响应速度，改进错误提示

## 技术亮点

- **创新交互**：自然语言控制，支持中英文，自动生成命令并解释
- **企业级安全**：生物识别 + SSH 密钥 + VPN 加密 + 危险命令识别
- **高性能架构**：数据持久化、全局状态管理、多设备并发连接
- **优秀体验**：聊天式界面、命令历史收藏、实时状态监控、友好错误提示
